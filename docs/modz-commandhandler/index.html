    A simple command handler for discord.js bots

    About

    modz-commandhandler is a simple to use command handler for discord.js.

    Installing

    Node 8.0.0 or higher required.

    To install it, run npm install modz-commandhandler in the console. It requires the master branch of discord.js, install it by running npm install hydrabolt/discord.js in the console.

    Git is required to install the master branch of discord.js

    Contributing

    If you want to contribute to modz-commandhandler, feel free to do so! Before doing that, please read the contribution guide.
