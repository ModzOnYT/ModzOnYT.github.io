    A simple command handler for discord.js bots

    About

    modz-commandhandler is a simple to use command handler for discord.js.

    Installing

    Node 8.0.0 or higher required! To install it, run npm install modz-commandhandler. It requires the master branch of discord.js, install it by running npm install hydrabolt/discord.js. Git is required to install the master branch of discord.js
