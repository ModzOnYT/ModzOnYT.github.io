ThunderStats

ThunderStats is a Discord bot made for use with the next-gen MMO combat game, War Thunder, written in JavaScript, built with the Nodejs library discord.js and the command framework Commando.

Functions

  • Viewing in-game squadron information and statistics;
  • Ability to check the latest news;
  • Search ability for the War Thunder Wiki.

In development

  • Viewing in-game player information and statistics;
  • Moderative tools using in-game information - useful for squadron servers.
